# Pascal's Triangle - Java

## Project Description
A simple program written in Java that calculates and prints Pascal's Triangle up to a user-specified number of levels. The program validates user input and includes safeguards to detect and handle integer overflows during calculations.

## Task description and requirements
[Task description (polish)](https://github.com/anna-julia-wojcik/Bsc-of-Bioinformatics/blob/main/Object-oriented-Design-and-Programming/pascalTriangle/task_requirements.txt)


## Project Structure

The project consists of one main file:

### 1. `Main.java`
- The main entry point of the program.
- Handles command-line arguments and input validation.
- Contains the logic to calculate, format, and print the rows of Pascal's Triangle.

## Requirements

- Java Development Kit (JDK) 8 or newer.

## How to Run the Program

Open a terminal, navigate to the project directory, compile the code, and run it by providing the number of levels as an argument:
```bash
cd pascals-triangle
javac Main.java
java Main 5
```
